== Description ==

Kale Pro is a clean, blog-focused theme perfect for food and lifestyle blogging. It has a custom frontpage settings to showcase special and featured posts in multiple areas with different formats. This is a perfect blogging theme with a multitude of features for customizing the layout.

* Front Page Banner
* Front Page Posts Slider		
* Front Page Featured Posts		
* Front Page Highlight Post		
* Image and Text Logos		
* Background Image		
* Background Color		
* Optional Full Width Posts and Pages		
* Special Category Template		
* Two-In-A-Row Blog Feed Layout		
* Mixed Format (Two Small Third Full) Blog Feed Layout		
* Integration with Recent Posts Widget With Thumbnails, MailPoet
* Easy to Create Social Media Icon Menus
* Header and Footer Area Widgets

* Front Page Custom Slider With Icons		
* Front Page Vertical Posts		
* Special Recipe Index Template		
* Built-in Social Media Sharing Icons (Using JetPack)		
* Built-in Recipe Shortcode		
* Built-in Ads
* Built-in Related Posts		
* Unlimited Color Choices		
* 600+ Google Fonts		
* Full Content/Image Blog Feed Layout		
* Google Analytics		
* Extra Page Post and Front Page Specific Sidebars		
* Extra Full Width Footer Widget Area (Instagram)	
* Integration with WP Instagram Widget	
* Structured Data Google Friendly Recipe Cards
